FONTES
FONT --------------- É POSSÍVEL UTILIZAR OS ATALHOS PARA DEFINIR TUDO EM UMA SÓ PROPRIEDADE
https://fonts.google.com/

font-family  --> família de fonte utilizada (ex: serif)

font-style --> estilo usado na fonte (ex: italic)

font-size  --> tamanho da fonte (ex: 12px)

font-weight  --> peso da fonte (ex: bold)

font-variant  --> altera a variante da fonte (ex: small-caps)

text-ident  --> recuo a esquerda da primeira linha (ex: 3em)

text-align  --> alinhamento do texto  (ex: left)

test-decoration  --> efeitos decorativos  (ex: underline)

world/lettter-spacing  --> espaçamento entre palavras/letras (ex: 1 em)

https://www.devmedia.com.br/css-background/38313

color  --> cor do texto (ex: #F3F3F3)

background-color  --> cor do fundo  (ex: black)

background-image  --> imagem de fundo (ex: url(http...))

background-repeat  --> define se a imagem de fundo repetirá  (ex: no repeat)

background-attachment  --> define se a imagem de fundo será fixa ou moverá conforme a página (ex: scroll)

background-position  --> posição da imagem de fundo (ex: right top)
